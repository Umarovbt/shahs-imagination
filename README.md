# Shah's Imagination

A one-page self-biography site for Shah, styled like a hand-drawn scrapbook — plain HTML and one CSS file, no build tools.

## Overview

- **Hero** — polaroid photo + intro tagline
- **About** — sticky-note bio blurb
- **What I Do** — index cards covering skills/interests
- **Hobbies & Interests** — sticker grid
- **Say Hi** — Instagram link styled as a postage stamp

## Structure

```
menu/
├── index.html      # Main page
├── style.css       # All styling (scrapbook/paper theme)
└── images/
    └── shah.jpg    # Profile photo
```

## Running locally

Just open `index.html` in your browser, or serve it with any static file server:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000`.
